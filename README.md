# 🎥 AI Gesture-Controlled Media Player

An interactive media player built using computer vision that allows users to control videos and music using **hand gestures and facial expressions in real-time**.

## 🚀 Features
- ✋ 5 Fingers → Start system
- 🖐️ 10 Fingers → Stop system
- 😊 Smile → Play/Pause music
- ☝️✌️🤟 1/2/3 Fingers → Switch between videos
- 📊 Real-time dashboard with gesture feedback
- 🎬 Dynamic video shapes (square, circle, triangle)

## 🛠️ Tech Stack
- Python
- OpenCV
- MediaPipe
- NumPy
- Pygame

## 💡 Highlights
- Real-time hand tracking and face detection
- Gesture-based UI control
- Smooth and responsive interaction system
- Creative dashboard for better user experience

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the project:
   python main.py

## ⚠️ Note
Add your own media files in the `assets` folder:
- clip.mp4
- clipp.mp4
- clippp.mp4
